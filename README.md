# ChessGameJavaFX

 Implemented feature:
 * Basic movement pattern
 * Low level rules restricted movement
 * Pawn's Promotion
 * Check and Checkmate condition
 * Pawn's En Passant special movement
 * King and Rook's Castling special movement
 * Save and load game
